# Vital-Eats
VitalEats is a simple front-end web application designed to promote healthy eating habits. The platform provides users with access to personalized meal plans, healthy recipes, and educational articles related to nutrition and wellness.

The website is built using HTML and CSS and features a structured layout with a navigation bar, side menu, and dynamic content display using an iframe. Users can easily navigate through different sections such as Home, About Us, Contact Us, Sign In, and Sign Up.

The goal of this project is to make healthy eating simple and accessible by presenting useful nutrition-related information in an organized and user-friendly interface.

Key Features

Clean and responsive UI design

Navigation bar for easy page access

Sidebar menu for content sections

Personalized meal plans section

Healthy recipes collection

Educational nutrition articles

Dynamic content loading using iframe

Technologies Used

HTML5

CSS3
